Welcome everyone! 
This is a basic, beginner friendly landing page created for open source contribution. You can find good first issues here. Let's Contribute!!

Steps To Contribute :
- **Step 1** - Fork this repository.
- **Step 2** - Clone the repository to your local machine.
- **Step 3** - Resolve the bugs, mentions provided in the Issues section of the repository. _Also add a description of what changes you have done_.
- **Step 4** - Add the changes to your repository.
- **Step 5** - Create a PULL Request. And that's all.
